Weather Swatch

What is Weather Swatch?

Need a little inspiration for the color scheme of your website?

Weather Swatch generates a collection of color swatches based on the current weather conditions in a specific city.

Under the Hood

This webapp was written using the BAR (Bootstrap + AngularJS + RequireJS) template I wrote which can be found here: https://github.com/JustinSYDE/BAR-Boilerplate. Tools used include:

  • AngularJS
  • AJAX
  • jQuery
  • Twitter Bootstrap
  • RequireJS
  • simpleWeather
