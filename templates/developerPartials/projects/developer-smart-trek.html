Fastest Game Setup

We worked with teachers and students in Washington to build a Notebook activity that allows teachers to setup a fun and educational multiplayer game in just a few seconds. This is achieved through our document parser that reads in teachers' multiple choice tests and outputs data for the game.

Flexibility

The game supports multiple themes and allows teachers to include different forms of media as answer options.

Under the Hood

Tools used include:

  • HTML, CSS, SASS
  • JavaScript, EmberJS
