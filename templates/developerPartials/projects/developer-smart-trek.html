Fastest Game Setup

My team and I worked with our target customers to build a Notebook activity that would allow teachers to setup a fun, educational, multiplayer game in just a few seconds. This was achieved with our document parser that could read in a teacher's multiple choice tests to populate data for the game.

Flexibility

The game supports multiple themes and also allows teachers to include different forms of media as answer options.
